# Port Status Checker

## Objective

The Port Status Checker is a Python-based networking tool that checks whether a specific port on a host is open or closed.

## Features

* Accepts a host name or IP address
* Accepts a port number as input
* Checks the status of the specified port
* Displays whether the port is OPEN or CLOSED
* Uses Python's socket module for network communication

## Technologies Used

* Python 3
* Socket Programming

## How It Works

1. The user enters a host name or IP address.
2. The user enters a port number.
3. The program attempts to establish a connection to the specified port.
4. If the connection is successful, the port is marked as OPEN.
5. If the connection fails, the port is marked as CLOSED.

## Sample Output

Enter host (Example: google.com or localhost): localhost

Enter port number: 80

Port 80 is CLOSED

## Applications

* Basic network troubleshooting
* Learning socket programming
* Checking service availability on a system
* Understanding network ports and connectivity
